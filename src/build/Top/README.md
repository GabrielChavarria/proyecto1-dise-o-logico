# Módulo principal del sistema

En esta carpeta se encuentra el módulo principal del sistema (top module).

Este módulo se encarga de integrar todos los subsistemas del proyecto,
incluyendo:

- Transmisor
- Receptor
- Módulos de visualización
- Conexión con los pines de la FPGA

El módulo top define las conexiones entre los diferentes bloques del
sistema y representa el punto de entrada del diseño para la síntesis
en la FPGA.

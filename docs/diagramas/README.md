# Diagramas del sistema

En esta carpeta se almacenan los diagramas de bloques del sistema digital
desarrollado en el Proyecto 1.

Los diagramas representan la estructura funcional de cada uno de los
subsistemas del transmisor y del receptor del sistema de corrección de
errores basado en el algoritmo Hamming (7,4).

Aquí se incluirán:

- Diagrama de bloques del transmisor
- Diagrama de bloques del receptor
- Diagramas de interconexión entre módulos
- Diagramas de subsistemas internos

Estos diagramas permiten visualizar la ruta de datos desde la entrada
de la palabra binaria hasta la salida corregida mostrada en los
dispositivos de visualización.

# Documentación del informe

En esta carpeta se almacenará el material utilizado para la elaboración
del informe final del proyecto.

El informe describe el funcionamiento completo del sistema digital
implementado, incluyendo el transmisor, el receptor y los mecanismos
de detección y corrección de errores mediante el código Hamming (7,4).

El informe incluye:

- Descripción general del sistema
- Explicación del funcionamiento de cada subsistema
- Diagramas de bloques del diseño
- Ejemplos de simplificación de ecuaciones booleanas
- Resultados de simulaciones
- Análisis del uso de recursos en la FPGA
- Problemas encontrados y soluciones aplicadas durante el desarrollo

# Testbench y verificación del sistema

Esta carpeta contiene los archivos de simulación utilizados para
verificar el funcionamiento de los diferentes módulos del sistema.

Los testbench permiten validar el comportamiento del diseño antes
de su implementación en la FPGA.

Se incluyen simulaciones para:

- Codificador Hamming
- Detector de errores
- Corrector de errores
- Funcionamiento completo del sistema

Las simulaciones se utilizan para observar las formas de onda y
comprobar que el sistema detecta y corrige errores correctamente.

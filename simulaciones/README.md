# Resultados de simulación

En esta carpeta se almacenan los resultados obtenidos durante las
simulaciones del sistema.

Aquí se guardarán:

- Capturas de formas de onda
- Archivos generados por las herramientas de simulación
- Resultados de pruebas funcionales

Estas simulaciones permiten analizar el comportamiento temporal de
las señales y validar el correcto funcionamiento del sistema antes
de su implementación en hardware.
